# 如何使用真实数据库数据

## 1. 确保MySQL数据库正在运行

请确保您的MySQL服务器正在运行，并且有以下配置：
- 主机地址: 127.0.0.1
- 端口: 3306
- 用户名: root
- 密码: pass2004
- 数据库名: jy_data_test1

如果您的配置不同，请修改 `server/.env` 文件中的相应设置。

## 2. 启动后端服务器

在命令行中进入服务器目录并安装依赖（如果尚未安装）：

```bash
cd server
npm install
```

然后启动服务器：

```bash
# 开发模式（自动重启）
npm run dev

# 或者生产模式
npm start
```

服务器将在 http://localhost:4000 上运行。

## 3. 启动前端应用

在另一个命令行窗口中，进入项目根目录并启动前端：

```bash
npm run dev
```

前端应用将在 http://localhost:5173 上运行（或类似地址）。

## 4. 验证连接

当前端应用加载后，它将通过代理连接到后端API (http://localhost:4000/api/*)，并从MySQL数据库获取真实数据。

如果您在浏览器控制台中看到 "API 失败，使用 mock 数据" 的警告，说明API请求失败了，请检查：
- 后端服务器是否正在运行
- 数据库连接是否正常
- 数据库中是否有数据

## 5. 故障排除

如果遇到问题，请检查：
- 后端服务器日志中的错误信息
- 浏览器开发者工具中的网络请求错误
- 数据库连接参数是否正确

您可以通过访问 http://localhost:4000/api/trains 直接测试API是否正常工作。
